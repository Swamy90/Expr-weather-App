

# Weather App

A simple Weather Application built using React Native.

## Features
- Search weather by city
- Real-time weather data
- Clean UI
- API integration

---

## Installation Steps

### 1. Clone the Repository
git clone https://github.com/Swamy90/Expr-weather-App.git

2. Navigate to Project Folder
cd Expr-weather-App

3. Install Dependencies
npm install

4. Start the Project
npx expo start
